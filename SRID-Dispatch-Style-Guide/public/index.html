Style Guide

11/28/2016

Summary

This dispatch communication application is created to support multiple channels for users to supply and obtain information. The target features include:

  1. Specify location using media regarding surrounding environments such as pictures
  2. Drop pin on map indicating the user's location to the best of their knowledge
  3. Automatically loop in emergency contacts on the user's status during emergencies
  4. Enable group calling instances with dispatch in wide spread instances
  5. Automated processes for standardized procedures such as towed cars

Buttons

Category Explanation Sample
Info These are non-significant buttons for page navigation purpose. They should demonstrate their functions properly but don't have to be very outstanding.
Confirm These buttons should appear in the middle of their pages.
Caution These are buttons that slightly catch more attention and will more easily attract user's attention in the case when he wants to interact with the application.
Danger These buttons are meant to be very eye catching and intuitive to cause pressing without much thought.
